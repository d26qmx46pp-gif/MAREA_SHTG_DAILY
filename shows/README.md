# Marea shows

Each subfolder is one podcast show created by the marea-podcast-maker skill.

Layout per show:
```
shows/{show-name}/
  feed.xml       Podcast RSS feed (subscribe URL)
  episodes/      MP3 audio files
```

Public feed URL: https://d26qmx46pp-gif.github.io/MAREA_SHTG_DAILY/shows/{show-name}/feed.xml

To add a new show, run the `marea-podcast-maker` skill in Cowork.
